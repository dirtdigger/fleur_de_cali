# Fleur de Cali
Tool for calibrating dimensionality and skew of a 3D printer.

This is a free and open source alternative to Califlower. Simply print the file, measure with your calipers, enter into the spreadsheet and calibrate your dimensions and skew!

The model is parametric in the sense that you can change the number of measurement points per axis in the source code. The spreadsheet is currently being modified to account for the parametric nature of the model and will be updated soon.
